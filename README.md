Приложение для учета рабочего времени в многомодульном варианте на Go.

Для запуска в докере:
1) Установить make
2) Выполнить `docker-compose up`

Для запуска бекенда локально:
1) Установить make
2) Выполнить `docker-compose -f docker-compose-local.yml up`
3) Выполнить `make migrate-all`
4) Выполнить `make run-all-local`
