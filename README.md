Приложение для учета рабочего времени в многомодульном варианте на Go.

Для запуска приложения в докере:
1) скачать репозиторий,
2) выполнить сборку микросервисов Employee, LogEntry, Api-gateway с помощью Maven (фронтенд не собирать),
3) запустить docker-compose

Для запуска локально:
1) Установить make
2) Выполнить `docker-compose -f docker-compose-local.yml up`
2) Выполнить `make run`
2) выполнить сборку микросервисов Employee, LogEntry, Api-gateway с помощью Maven (фронтенд не собирать),
3) запустить docker-compose
